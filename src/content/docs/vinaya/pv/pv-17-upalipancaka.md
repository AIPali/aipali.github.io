> [!abstract] AI 导读
> 本篇名为《优波离问五法》，是巴利律藏《附随》中的重要篇章。全文以问答形式构成，由被誉为“持律第一”的优波离尊者，就僧团运作和个人修持中遇到的具体戒律问题，向佛陀提问。佛陀的回答则系统地将复杂的戒律议题归纳为五项一组的原则，因此得名“五法”。
>
> 内容涵盖了比丘何时可以独立修行（无需依止师）、授戒师的资格、应受僧团处分（羯磨）的条件、处理僧团争端（灭诤）、分裂僧团（僧残十、十一）的成因与后果、对不同罪行的界定，以及对比丘尼的教诫规范等诸多方面。
>
> 本篇不仅是研究早期佛教僧团制度和戒律精神的珍贵文献，其严谨的法理逻辑和对僧团清净、和合的高度重视，至今仍对僧团的健康运作具有重要的指导意义。

## 1 依止品 (Anissitavagga)

### 1.1 何时可以无需依止

我是这样听闻的：有一次，世尊住在舍卫城祇树给孤独园。那时，尊者优波离来到世尊跟前，顶礼世尊后，坐在一旁。坐定后，尊者优波离向世尊请教：“尊者，一位比丘需要具备哪几项条件，才能终身无需依止师而独立安住呢？”[P:417]

“优波离，一位比丘若具备五项条件，就必须终身依止他人，不能独立安住。是哪五项呢？

1. 不了解什么是布萨 (Uposatha)。
2. 不了解如何进行布萨羯磨（僧团决议）。
3. 不了解波罗提木叉 (Pātimokkha) 的内容。
4. 不了解如何诵念波罗提木叉。
5. 戒龄未满五年。

优波离，具备这五项条件的比丘，必须终身依止他人，不能独立安住。

优波离，一位比丘若具备另外五项条件，就可以终身无需依止师而独立安住。是哪五项呢？

1. 了解什么是布萨。
2. 了解如何进行布萨羯磨。
3. 了解波罗提木叉的内容。
4. 了解如何诵念波罗提木叉。
5. 戒龄已满五年或超过五年。

优波离，具备这五项条件的比丘，就可以终身无需依止师而独立安住。

优波离，又另有五项条件，具备这些的比丘必须终身依止他人，不能独立安住。是哪五项呢？

1. 不了解什么是自恣 (Pavāraṇā)。
2. 不了解如何进行自恣羯磨。
3. 不了解波罗提木叉的内容。
4. 不了解如何诵念波罗提木叉。
5. 戒龄未满五年。

优波离，具备这五项条件的比丘，必须终身依止他人，不能独立安住。

优波离，一位比丘若具备另外五项条件，就可以终身无需依止师而独立安住。是哪五项呢？

1. 了解什么是自恣。
2. 了解如何进行自恣羯磨。
3. 了解波罗提木叉的内容。
4. 了解如何诵念波罗提木叉。
5. 戒龄已满五年或超过五年。

优波离，具备这五项条件的比丘，就可以终身无需依止师而独立安住。

优波离，又另有五项条件，具备这些的比丘必须终身依止他人，不能独立安住。是哪五项呢？

1. 不知何为有罪，何为无罪。
2. 不知何为轻罪，何为重罪。
3. 不知何为有残罪（即僧残等可忏罪），何为无残罪（即弃罪等不可忏罪）。
4. 不知何为粗罪，何为非粗罪。
5. 戒龄未满五年。

优波离，具备这五项条件的比丘，必须终身依止他人，不能独立安住。

优波离，一位比丘若具备另外五项条件，就可以终身无需依止师而独立安住。是哪五项呢？

1. 知晓何为有罪，何为无罪。
2. 知晓何为轻罪，何为重罪。
3. 知晓何为有残罪，何为无残罪。
4. 知晓何为粗罪，何为非粗罪。
5. 戒龄已满五年或超过五年。

优波离，具备这五项条件的比丘，就可以终身无需依止师而独立安住。”

### 1.2 何人不可授戒或接收弟子

“尊者，一位比丘具备哪几项条件，就不可以为他人授具足戒、给予他人依止、或接收沙弥为弟子呢？”[P:418]

“优波离，具备五项条件的比丘，不可以为他人授具足戒、给予他人依止、或接收沙弥为弟子。是哪五项呢？

1. 当弟子或共住者生病时，没有能力亲自照顾，也没有能力安排他人照顾。
2. 当弟子或共住者对修行生活产生厌烦时，没有能力劝导他，也没有能力安排他人劝导他。
3. 当弟子或共住者心中生起疑悔时，没有能力依据正法为他消除。
4. 不能教导阿毗达摩（论）。
5. 不能教导阿毗毗奈耶（广律）。

优波离，具备这五项条件的比丘，不可以为他人授具足戒、给予他人依止、或接收沙弥为弟子。

优波离，具备另外五项条件的比丘，可以为他人授具足戒、给予他人依止、或接收沙弥为弟子。是哪五项呢？

1. 当弟子或共住者生病时，有能力亲自照顾，或安排他人照顾。
2. 当弟子或共住者对修行生活产生厌烦时，有能力劝导他，或安排他人劝导他。
3. 当弟子或共住者心中生起疑悔时，有能力依据正法为他消除。
4. 能够教导阿毗达摩。
5. 能够教导阿毗毗奈耶。

优波离，具备这五项条件的比丘，可以为他人授具足戒、给予他人依止、或接收沙弥为弟子。

优波离，又另有五项条件，具备这些的比丘不可以为他人授具足戒、给予他人依止、或接收沙弥为弟子。是哪五项呢？

1. 不能教导弟子或共住者有关增上行仪的学处。
2. 不能教导有关初梵行的学处。
3. 不能教导增上戒学。
4. 不能教导增上心学。
5. 不能教导增上慧学。

优波離，具备这五项条件的比丘，不可以为他人授具足戒、给予他人依止、或接收沙弥为弟子。

优波离，具备另外五项条件的比丘，可以为他人授具足戒、给予他人依止、或接收沙弥为弟子。是哪五项呢？

1. 能够教导弟子或共住者有关增上行仪的学处。
2. 能够教导有关初梵行的学处。
3. 能够教导增上戒学。
4. 能够教导增上心学。
5. 能够教导增上慧学。

优波离，具备这五项条件的比丘，可以为他人授具足戒、给予他人依止、或接收沙弥为弟子。”

> [!note] AI 解析：关于依止 (Nissaya)
> “依止”是佛教僧团中一项非常重要的制度，指的是新比丘必须跟随一位具格的、有经验的长老比丘（依止师）学习戒律和法义，至少五年。这种师徒关系确保了教法的准确传承和僧人行为的规范。本段经文明确了“出依止”（即不再需要依止师）的资格，核心在于对戒律的精通和足够的戒龄，这是僧团教育成熟度的标志。

### 1.3 应对何人进行僧团处分

“尊者，对具备哪几项条件的比丘，僧团应当对他进行羯磨（即采取纪律处分）呢？”[P:419]

“优波离，对具备五项条件的比丘，僧团应当对他进行羯磨。是哪五项呢？

1. 无耻。
2. 愚痴。
3. 行为不清净。
4. 持有邪见。
5. 生计腐败（邪命）。

优波离，对具备这五项条件的比丘，僧团应当对他进行羯磨。

优波离，又另有五项条件，对具备这些的比丘，僧团应当对他进行羯磨。是哪五项呢？

1. 在增上戒方面，戒行败坏。
2. 在日常行为方面，品行败坏。
3. 在正见方面，见解败坏。
4. 持有邪见。
5. 生计腐败。

优波离，对具备这五项条件的比丘，僧团应当对他进行羯磨。

优波离，又另有五项条件，对具备这些的比丘，僧团应当对他进行羯磨。是哪五项呢？

1. 具有嬉戏的身业。
2. 具有嬉戏的语业。

优波离，对具备这五项条件的比丘，僧团应当对他进行羯磨。

## 2 不解羯磨品 (Nappaṭippassambhanavagga)

### 2.1 不得解除羯磨的条件

“尊者，对具备哪几项条件的比丘，僧团不应当为他解除羯磨处分呢？”[P:420]

“优波离，对具备五项条件的比丘，僧团不应当为他解除羯磨处分。是哪五项呢？

1. 在已被僧团羯磨处分期间，还为他人授具足戒。
2. 给予他人依止。
3. 接收沙弥为弟子。
4. 同意接受未经选派教诫比丘尼的任命。
5. 在接受任命后，去教诫比丘尼。

优波离，对具备这五项条件的比丘，僧团不应当为他解除羯磨处分。

优波离，又另有五项条件，对具备这些的比丘，僧团不应当为他解除羯磨处分。是哪五项呢？

1. 重复犯下导致他被僧团羯磨的那个罪行。
2. 犯下其他同类的罪行。
3. 犯下比原来更严重的罪行。
4. 斥责此羯磨本身。
5. 斥责执行羯磨的人。

优波离，对具备这五项条件的比丘，僧团不应当为他解除羯磨处分。

优波离，又另有五项条件，对具备这些的比丘，僧团不应当为他解除羯磨处分。是哪五项呢？

1. 诽谤佛。
2. 诽谤法。
3. 诽谤僧。
4. 持有邪见。
5. 生计腐败。

优波离，对具备这五项条件的比丘，僧团不应当为他解除羯磨处分。

优波离，又另有五项条件，对具备这些的比丘，僧团不应当为他解除羯磨处分。是哪五项呢？

1. 无耻。
2. 愚痴。
3. 行为不清净。
4. 在各种事务中行为粗暴。
5. 未圆满完成应学的学处。

优波离，对具备这五项条件的比丘，僧团不应当为他解除羯磨处分。”

### 2.2 参与僧团议事应具备的心态

“尊者，一位需要处理争端的比丘，在前往僧团集会时，内心应当具备哪几项法才能前往呢？”[P:421]

“优波离，一位需要处理争端的比丘，在前往僧团集会时，内心应当具备五项法才能前往。是哪五项呢？

1. 应当怀着谦下之心进入僧团，心态要如同擦拭灰尘的抹布一样。
2. 应当善于选择和谦让座位，不挤到长老比丘的座位，也不排挤年轻比丘的座位，应坐在适合自己的位置上。
3. 不应说前后矛盾或无关的话题，应当自己说法，或请他人说法，对于圣者的沉默不应轻视。
4. 优波离，如果僧团正在进行需要全体和合才能完成的羯磨，此时若有比丘持不同意见，也应当表明观点，以求达成僧团的和合一致。为何如此？是为了‘我与僧团没有分歧’。
5. 优波离，一位需要处理争端的比丘，在前往僧团集会时，内心应当具备这五项法才能前往。”

### 2.3 僧团中不受欢迎与受欢迎的行为

“尊者，具备哪几项条件的比丘，在僧团中处理事务时，会令大众不喜、不爱、不乐呢？”[P:422]

“优波离，具备五项条件的比丘，在僧团中处理事务时，会令大众不喜、不爱、不乐。是哪五项呢？

1. 心怀恶意地发言。
2. 依附他人而发言。
3. 不善于运用语言。
4. 对于应依法、依律、依罪呵责的人，不予以呵责。
5. 对于应依法、依律、依罪处理的人，不予以处理。

优波离，具备这五项条件的比丘，在僧团中处理事务时，会令大众不喜、不爱、不乐。

优波离，具备另外五项条件的比丘，在僧团中处理事务时，会令大众欢喜、喜爱、乐见。是哪五项呢？

1. 不心怀恶意地发言。
2. 不依附他人而发言。
3. 善于运用语言。
4. 对于应依法、依律、依罪呵责的人，予以呵责。
5. 对于应依法、依律、依罪处理的人，予以处理。
优波离，具备这五项条件的比丘，在僧团中处理事务时，会令大众欢喜、喜爱、乐见。

优波离，又另有五项条件，具备这些的比丘在僧团中处理事务时，会令大众不喜、不爱、不乐。是哪五项呢？

1. 赞扬他人（不当的）。
2. 贬斥他人。
3. 执着于非法。
4. 抗拒正法。
5. 言语多而杂乱无益。

优波离，具备这五项条件的比丘，在僧团中处理事务时，会令大众不喜、不爱、不乐。

优波离，具备另外五项条件的比丘，在僧团中处理事务时，会令大众欢喜、喜爱、乐见。是哪五项呢？

1. 不（不当地）赞扬他人。
2. 不贬斥他人。
3. 执着于正法。
4. 抗拒非法。
5. 言语不多而无杂乱。

优波离，具备这五项条件的比丘，在僧团中处理事务时，会令大众欢喜、喜爱、乐见。

优波离，又另有五项条件，具备这些的比丘在僧团中处理事务时，会令大众不喜、不爱、不乐。是哪五项呢？

1. 强行推进事务。
2. 未获得许可就推进事务。
3. 对于应依法、依律、依罪呵责的人，不予以呵责。
4. 对于应依法、依律、依罪处理的人，不予以处理。

### 5.1 呵责他人的资格

### 5.5 自行处理事务的审查标准

“尊者，一位想自行处理某项事务（attādāna）的比丘，应当具备哪几项条件，才能自行处理？”[P:440]

“优波离，一位想自行处理某项事务的比丘，应当审查五项条件后，才能自行处理。是哪五项呢？

1. 他应反省：‘我想处理的这件事，现在是合适的时机吗？’如果反省后，他知道现在不是合适的时机，那么就不应处理。
2. 如果他知道现在是合适的时机，应进一步反省：‘我想处理的这件事，是基于事实的吗？’如果反省后，他知道此事并非基于事实，那么就不应处理。
3. 如果他知道此事是基于事实的，应进一步反省：‘我想处理的这件事，是有益的吗？’如果反省后，他知道此事是无益的，那么就不应处理。
4. 如果他知道此事是有益的，应进一步反省：‘我处理这件事时，能否得到正直、可敬的比丘们在法和律上的支持？’如果反省后，他知道无法得到支持，那么就不应处理。
5. 如果他知道能得到支持，应进一步反省：‘我处理这件事，是否会因此在僧团中引起争论、冲突、纠纷、分裂僧团、不和、或各种分歧？’如果反省后，他知道会引起僧团不和，那么就不应处理。

如果他知道不会引起僧团不和，那么这件事就可以处理。优波离，像这样具备了五项条件的审查后所处理的事务，将来才不会引起追悔。”

4. 不知晓如何平息罪行。
5. 不善于决断罪行。

优波离，具备这五项条件的比丘，没有能力平息争端。

优波离，具备另外五项条件的比丘，有能力平息争端。是哪五项呢？

1. 知晓罪行。
2. 知晓罪行的起因。
3. 知晓罪行的具体行为。
4. 知晓如何平息罪行。
5. 善于决断罪行。

优波离，具备这五项条件的比丘，有能力平息争端。

（接下来，佛陀以同样的结构，重复了七组判断是否有能力平息争端的标准）：

- **第二组：** 不知晓争端、起因、过程、平息、不善决断者，无能力；反之，有能力。
- **第三组：** 随贪、嗔、痴、怖而行，且无惭愧者，无能力；反之，有能力。
- **第四组：** 随贪、嗔、痴、怖而行，且少闻者，无能力；反之（不随四不善行且多闻者），有能力。
- **第五组：** 不知晓事由、因缘、制戒内容、文句念诵、前后关联者，无能力；反之，有能力。
- **第六组：** 随贪、嗔、痴、怖而行，且不善巧于戒律者，无能力；反之，有能力。
- **第七组：** 随贪、嗔、痴、怖而行，且重人情而不重僧团者，无能力；反之（重僧团不重人情者），有能力。
- **第八组：** 随贪、嗔、痴、怖而行，且重利养而不重正法者，无能力；反之（重正法不重利养者），有能力。

优波离，具备这后五项条件的比丘，有能力平息争端。”

### 10.2 僧团分裂的方式与界定

“尊者，僧团会因哪几种方式而分裂？”[P:458]

“优波离，僧团会因五种方式而分裂。是哪五项呢？

1. 通过羯磨。
2. 通过诵念。
3. 通过决断事务。
4. 通过宣告。
5. 通过抽签表决。

优波离，僧团会因这五种方式而分裂。”

“尊者，常说‘僧诤’，那么，在什么情况下是僧诤，但还不是分裂僧团（僧残十）？又在什么情况下既是僧诤，也是分裂僧团呢？”
“优波离，我为外来比丘制定了客住仪法。优波离，在我善巧制定了这些学处后，如果外来比丘不遵守客住仪法，优波离，这样就是僧诤，但还不是分裂僧团。

优波离，我为本地比丘制定了本地仪法。优波离，在我善巧制定了这些学处后，如果本地比丘不遵守本地仪法，优波离，这样就是僧诤，但还不是分裂僧团。

优波离，我为诸比丘制定了食堂仪法：应按戒腊长幼、年龄老少、以及是否适当来分配上座、净水和食物。优波离，在我善巧制定了这些学处后，如果新比丘在食堂强占长老比丘的座位，优波离，这样就是僧诤，但还不是分裂僧团。

优波离，我为诸比丘制定了住宿仪法：应按戒腊长幼、年龄老少、以及是否适当来分配。优波离，在我善巧制定了这些学处后，如果新比丘抢占长老比丘的住处，优波离，这样就是僧诤，但还不是分裂僧团。

优波离，我为诸比丘制定了在同一界内，应行同一布萨、同一自恣、同一僧伽羯磨、同一羯磨与非羯磨的规定。优波离，在我善巧制定了这些学处后，如果在同一界内，有人另立门户，集结自己的团体，举行独立的布萨、独立的自恣、独立的僧伽羯磨、独立的羯磨与非羯磨，优波离，这样就既是僧诤，也是分裂僧团了。”

## 11 分裂僧团者品 (Saṅghabhedakavagga)

### 11.1 导致堕地狱的分裂僧团行为

“尊者，具备哪几项条件的分裂僧团者，会堕入恶道、堕入地狱，长住一劫而无法救治？”[P:459]

“优波离，具备五项条件的分裂僧团者，会堕入恶道、堕入地狱，长住一劫而无法救治。是哪五项呢？

1. 优波离，在此，有比丘将非法说成是法。
2. 将法说成是非法。
3. 将非律说成是律。
4. 将律说成是非律。
5. 并另立自己的见解，通过羯磨来分裂僧团。

优波离，具备这五项条件的分裂僧团者，会堕入恶道、堕入地狱，长住一劫而无法救治。

（接下来，佛陀以同样的结构，指出了另外三组导致同样严重后果的分裂僧团行为，其核心都是“将非法说成法、法说成非法、非律说成律、律说成非律”，不同之处在于分裂僧团的具体方式）：

- **第二组：** ……并另立自己的见解，通过诵念来分裂僧团。
- **第三组：** ……并另立自己的见解，通过决断事务来分裂僧团。
- **第四组：** ……并另立自己的见解，通过宣告来分裂僧团。
- **第五组：** ……并另立自己的见解，通过抽签表决来分裂僧团。

（佛陀接着又列举了三类，每类包含五种方式，共十五种组合，其核心是除了“另立见解”，还包括“另立忍可”、“另立喜好”、“另立想法”，并通过上述五种方式来分裂僧团。这些行为都会导致同样的严重后果。）

优波离，具备这些条件的分裂僧团者，会堕入恶道、堕入地狱，长住一劫而无法救治。”

## 12 第二分裂僧团者品 (Dutiyasaṅghabhedakavagga)

### 12.1 不导致堕地狱的分裂僧团行为

“尊者，具备哪几项条件的分裂僧团者，不会堕入恶道、不会堕入地狱，不会长住一劫，也并非无法救治？”[P:460]

“优波离，具备五项条件的分裂僧团者，不会堕入恶道、不会堕入地狱，不会长住一劫，也并非无法救治。是哪五项呢？

1. 优波离，在此，有比丘将非法说成是法。
2. 将法说成是非法。
3. 将非律说成是律。
4. 将律说成是非律。
5. 但他并非为了另立自己的见解而通过羯磨来分裂僧团。

优波离，具备这五项条件的分裂僧团者，不会堕入恶道……并非无法救治。

（与前一品类似，佛陀以同样的结构，阐述了虽然行为上导致了僧团分裂，但由于动机并非出于恶意地另立邪见、邪忍、邪好、邪想，因此其果报与前者有天壤之别。这些组合包括）：

- 不为另立见解，而通过诵念、决断、宣告、抽签分裂僧团。
- 不为另立忍可，而通过羯磨、诵念、决断、宣告、抽签分裂僧团。
- 不为另立喜好，而通过羯磨、诵念、决断、宣告、抽签分裂僧团。
- 不为另立想法，而通过羯磨、诵念、决断、宣告、抽签分裂僧团。

优波离，具备这些条件的分裂僧团者，不会堕入恶道、不会堕入地狱，不会长住一劫，也并非无法救治。”

## 14 迦絺那衣品 (Kathinatthāravagga)
