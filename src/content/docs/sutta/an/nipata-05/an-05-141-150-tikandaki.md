## 1 AN 5.141 五种轻蔑经 (Avajānātisutta)

世尊说：“诸比丘！世界上有这五种人存在。是哪五种呢？”[P:141]

1. 布施后轻蔑他人。
2. 共住后轻蔑他人。
3. 轻率听信他人。
4. 摇摆不定（信仰微弱）。
5. 愚钝痴呆。

“诸比丘！怎样是布施后轻蔑他人呢？诸比丘！世间有人将衣服、饮食、住所、医药等必需品布施给他人。他心里想：‘是我布施给他，他只是接受而已。’因此，在布施之后，他就轻蔑对方。诸比丘！这就是布施后轻蔑他人。

“怎样是共住后轻蔑他人呢？诸比丘！世间有人与他人共同生活两年或三年。因为长久共住的缘故，他就轻蔑对方。诸比丘！这就是共住后轻蔑他人。

“怎样是轻率听信他人呢？诸比丘！世间有的人，一听到别人受到赞叹或毁谤，就立刻深信不疑。诸比丘！这就是轻率听信他人。

“怎样是摇摆不定呢？诸比丘！世间有的人，信仰微弱，恭敬心微弱，慈爱微弱，净信微弱。诸比丘！这就是摇摆不定。

“怎样是愚钝痴呆呢？诸比丘！世间有的人，不知道什么是善法、什么是不善法，不知道什么是有罪之法、什么是无罪之法，不知道什么是低劣之法、什么是殊胜之法，不知道什么是黑白相对之法。诸比丘！这就是愚钝痴呆。

“诸比丘！世界上有这五种人存在。”

## 2 AN 5.142 违犯之行经 (Ārabhatisutta)

> [!note] AI 解析：违犯戒律与追悔的心理机制
> 本经深入探讨了修行者的两种常见心理障碍：“违犯学处”与“事后追悔（内疚）”。佛陀指出，即使造作了违犯之行，过度的懊悔（疑神疑鬼）不仅无济于事，反而会成为新的烦恼漏（掉举恶作盖）。正确的做法是：认清过失，放下沉重的内疚感，通过如实了知并修习定慧（心解脱与慧解脱），将恶法彻底灭尽。

“诸比丘！世界上有这五种人存在。是哪五种呢？”[P:142]

1. 违犯戒律并且产生追悔，且不如实了知心解脱与慧解脱——然而在这心解脱与慧解脱中，他们生起的恶不善法本能无余地灭尽。
2. 违犯戒律但不产生追悔，且不如实了知心解脱与慧解脱——然而在这心解脱与慧解脱中，他们生起的恶不善法本能无余地灭尽。
3. 没有违犯戒律但产生追悔，且不如实了知心解脱与慧解脱——然而在这心解脱与慧解脱中，他们生起的恶不善法本能无余地灭尽。
4. 没有违犯戒律也不产生追悔，且不如实了知心解脱与慧解脱——然而在这心解脱与慧解脱中，他们生起的恶不善法本能无余地灭尽。
5. 没有违犯戒律也不产生追悔，并且如实了知心解脱与慧解脱——在这心解脱与慧解脱中，他们生起的恶不善法已无余地灭尽。

“诸比丘！这当中，对于那位违犯戒律并且产生追悔，且不如实了知心解脱与慧解脱的人，应当这样对他说：‘尊者，由违犯戒律而产生的漏烦恼依然存在，由追悔而产生的漏烦恼还在增长。尊者最好能断除由违犯戒律而产生的漏烦恼，排除由追悔而产生的漏烦恼，从而勤加修习心与智慧。如果能这样做，尊者就完全可以与第五种人一样。’

“诸比丘！这当中，对于那位违犯戒律但不产生追悔，且不如实了知心解脱与慧解脱的人，应当这样对他说：‘尊者，由违犯戒律而产生的漏烦恼依然存在，但由追悔而产生的漏烦恼没有增长。尊者最好能断除由违犯戒律而产生的漏烦恼，从而勤加修习心与智慧。如果能这样做，尊者就完全可以与第五种人一样。’

“诸比丘！这当中，对于那位没有违犯戒律但产生追悔，且不如实了知心解脱与慧解脱的人，应当这样对他说：‘尊者，由违犯戒律而产生的漏烦恼并不存在，但由追悔而产生的漏烦恼在增长。尊者最好能排除由追悔而产生的漏烦恼，从而勤加修习心与智慧。如果能这样做，尊者就完全可以与第五种人一样。’

“诸比丘！这当中，对于那位没有违犯戒律也不产生追悔，且不如实了知心解脱与慧解脱的人，应当这样对他说：‘尊者，由违犯戒律而产生的漏烦恼并不存在，由追悔而产生的漏烦恼也没有增长。尊者最好能勤加修习心与智慧。如果能这样做，尊者就完全可以与第五种人一样。’

“诸比丘！前四种人如果被第五种人这样教导、指示，就能逐渐达到诸漏已尽的境界。”

## 3 AN 5.143 娑罗达达经 (Sārandadasutta)

我是这样听闻的：有一次，世尊住在毗舍离的大林重阁讲堂。[P:143]

那天上午，世尊穿好内衣，拿着钵与袈裟，进入毗舍离城托钵乞食。当时，大约有五百位离车人聚集在娑罗达达塔前，谈论着这样的话题：“世间有五种宝物很难出现。是哪五种呢？世间难得出现象宝，世间难得出现马宝，世间难得出现摩尼宝，世间难得出现女宝，世间难得出现居士宝。世间难得出现这五种宝物。”

这时，离车人在路上安排了一个人，对他说：“喂，朋友！如果你看到世尊走过来，就立刻告诉我们。”那个人远远看到世尊走来，便赶紧去告诉离车人们：“各位尊者！那位世尊、阿罗汉、正等正觉者正朝这边走来，你们看现在是不是时候去拜见他？”

于是，离车人们来到世尊面前，向世尊顶礼，然后站在一旁。站在一旁的离车人们对世尊说：“世尊！如果世尊能出于哀悯，前往娑罗达达塔，那就太好了。”世尊默然答应了。

随后，世尊前往娑罗达达塔，坐在为他准备好的座位上。坐下后，世尊问离车人们：“离车人们！你们刚才聚集在这里讨论什么话题？刚才未谈完的话题是什么？”

“世尊！我们刚才聚集在这里，谈论说：‘世间有五种宝物很难出现。是哪五种呢？象宝、马宝、摩尼宝、女宝、居士宝难得出现。世间难得出现这五种宝物。’”

“离车人们！沉迷于五欲的离车人，果然只会谈论关于五欲的话题。离车人们！世间真正有五种宝物是很难出现的。是哪五种呢？

1. 如来、阿罗汉、正等正觉者在世间难得出现。
2. 能解说如来所宣示的法与律的人，在世间难得出现。
3. 能理解并领悟如来所宣示的法与律的人，在世间难得出现。
4. 在理解如来所宣示的法与律后，能够依法修行的人（法随法行者），在世间难得出现。
5. 知恩、感恩的人，在世间难得出现。

“离车人们！世间难得出现这五种真正的宝物。”

## 4 AN 5.144 底甘陀林经 (Tikaṇḍakīsutta)

> [!example] AI 譬喻解析：圣者的神变 (Ariya Iddhi)
> 经中描述的这五种心态训练，在南传法义中被称为“圣者的神变”或“圣者的威力”。它不是指呼风唤雨的特异功能，而是内心对外界刺激的绝对掌控力。普通人面对悦意境生贪、对违意境生瞋，而修行者能够通过转换认知（如在美食前作不净观，在逆境中修慈心），随意调整自己的心理反应，最终安住于超越两极的平等舍心。

有一次，世尊住在娑鸡帝的底甘陀林。[P:144]

在那里，世尊对诸比丘说：“诸比丘！”比丘们回答世尊：“大德！”世尊这样说道：

“善哉，诸比丘！比丘应当安住于不厌恶（愉快）的对境中，时时作厌恶想。\
善哉，诸比丘！比丘应当安住于厌恶（讨厌）的对境中，时时作不厌恶想。\
善哉，诸比丘！比丘应当安住于不厌恶和厌恶的对境中，时时作厌恶想。\
善哉，诸比丘！比丘应当安住于厌恶和不厌恶的对境中，时时作不厌恶想。\
善哉，诸比丘！比丘应当避开厌恶与不厌恶这两者，保持平等心（舍心），保持正念、正知。

“诸比丘！基于什么目的，比丘应当安住于不厌恶的对境中，时时作厌恶想呢？他思惟：‘愿我不要对令人贪爱的对境生起贪爱。’诸比丘！正是基于这个目的，比丘应当安住于不厌恶的对境中，时时作厌恶想。

“诸比丘！基于什么目的，比丘应当安住于厌恶的对境中，时时作不厌恶想呢？他思惟：‘愿我不要对令人瞋恨的对境生起瞋恨。’诸比丘！正是基于这个目的，比丘应当安住于厌恶的对境中，时时作不厌恶想。

“诸比丘！基于什么目的，比丘应当安住于不厌恶和厌恶的对境中，时时作厌恶想呢？他思惟：‘愿我不要对令人贪爱的对境生起贪爱，也不要对令人瞋恨的对境生起瞋恨。’诸比丘！正是基于这个目的，比丘应当安住于不厌恶和厌恶的对境中，时时作厌恶想。

“诸比丘！基于什么目的，比丘应当安住于厌恶和不厌恶的对境中，时时作不厌恶想呢？他思惟：‘愿我不要对令人瞋恨的对境生起瞋恨，也不要对令人贪爱的对境生起贪爱。’诸比丘！正是基于这个目的，比丘应当安住于厌恶和不厌恶的对境中，时时作不厌恶想。

“诸比丘！基于什么目的，比丘应当避开厌恶与不厌恶这两者，保持平等心（舍心），保持正念、正知呢？他思惟：‘愿我在任何地方、任何时候、面对任何事物，都不要对令人贪爱的对境生起贪爱；在任何地方、任何时候、面对任何事物，都不要对令人瞋恨的对境生起瞋恨；在任何地方、任何时候、面对任何事物，都不要对令人迷惑的对境生起愚痴。’诸比丘！正是基于这个目的，比丘应当避开厌恶与不厌恶这两者，保持平等心（舍心），保持正念、正知。”

## 5 AN 5.145 地狱五法经 (Nirayasutta)

“诸比丘！具备这五种法的人，就如同被押送一般，死后必将堕入地狱。是哪五种呢？”[P:145]

1. 杀生。
2. 不与取（偷盗）。
3. 邪淫（欲邪行）。
4. 虚妄语。
5. 饮用会导致放逸的谷酒、果酒等酒精饮料。

“诸比丘！具备这五种法的人，就如同被押送一般，死后必将堕入地狱。

“诸比丘！具备这五种法的人，就如同被迎请一般，死后必将升入天界。是哪五种呢？

1. 远离杀生。
2. 远离不与取（偷盗）。
3. 远离邪淫（欲邪行）。
4. 远离虚妄语。
5. 远离会导致放逸的谷酒、果酒等酒精饮料。

“诸比丘！具备这五种法的人，就如同被迎请一般，死后必将升入天界。”

## 6 AN 5.146 亲近五法经 (Mittasutta)

“诸比丘！具备五种法的比丘，不应当被作为善友来亲近。是哪五种呢？”[P:146]

1. 喜欢从事世俗事务（如农业、建造等营生）。
2. 喜欢卷入诤端和诉讼。
3. 与上座比丘们对立、对抗。
4. 经常漫无目的地长途游荡。
5. 没有能力时常以佛法来开导、鼓励、激发、令他人欢喜。

“诸比丘！具备这五种法的比丘，不应当被作为善友来亲近。

“诸比丘！具备五种法的比丘，应当被作为善友来亲近。是哪五种呢？

1. 不从事世俗事务。
2. 不卷入诤端和诉讼。
3. 不与上座比丘们对立、对抗。
4. 不漫无目的地长途游荡。
5. 有能力时常以佛法来开导、鼓励、激发、令他人欢喜。

“诸比丘！具备这五种法的比丘，应当被作为善友来亲近。”

## 7 AN 5.147 非善士施经 (Asappurisadānasutta)

“诸比丘！有这五种非善士的布施（不善人的布施）。是哪五种呢？”[P:147]

1. 不恭敬地布施。
2. 不用心（无诚意）地布施。
3. 不亲手布施。
4. 把像是被抛弃的物品布施出去。
5. 不相信业果法则（不认为布施有果报）地布施。

“诸比丘！这就是五种非善士的布施。

“诸比丘！有这五种善士的布施（善人的布施）。是哪五种呢？

1. 恭敬地布施。
2. 用心（诚意）地布施。
3. 亲手布施。
4. 不把像是被抛弃的物品布施出去。
5. 相信业果法则（相信布施有果报）地布施。

“诸比丘！这就是五种善士的布施。”

## 8 AN 5.148 五善士施经 (Sappurisadānasutta)

“诸比丘！有这五种善士的布施。是哪五种呢？”[P:148]

1. 带着信仰布施。
2. 恭敬地布施。
3. 在适当的时机布施。
4. 以宽宏不吝啬的心（慈悲心）布施。
5. 在不伤害自己和他人的情况下布施。

“诸比丘！因为带着信仰布施，无论这布施的果报在哪里成熟，他都会变得富有、拥有大量财富和丰厚的受用，并且相貌端严、令人心生净信、具足殊胜的容颜。

“诸比丘！因为恭敬地布施，无论这布施的果报在哪里成熟，他都会变得富有、拥有大量财富和丰厚的受用。凡是他的妻子、儿女、仆人、使者、雇工等眷属，都会乐意恭顺、专心倾听，并且用心去理解他的意思。

“诸比丘！因为在适当的时机布施，无论这布施的果报在哪里成熟，他都会变得富有、拥有大量财富和丰厚的受用，并且各种利益和需求都能适时、丰沛地得到满足。

“诸比丘！因为以宽宏不吝啬的心布施，无论这布施的果报在哪里成熟，他都会变得富有、拥有大量财富和丰厚的受用，并且他的心会倾向于享受殊胜的五欲功德（五种感官之乐）。

“诸比丘！因为在不伤害自己和他人的情况下布施，无论这布施的果报在哪里成熟，他都会变得富有、拥有大量财富和丰厚的受用，并且他的财产不会因为火灾、水灾、王室没收、盗贼抢劫或不肖的继承人而受到损失。

“诸比丘！这就是五种善士的布施。”

## 9 AN 5.149 时解脱者之一经 (Paṭhamasamayavimuttasutta)

>[!caution] AI 区分：此处的“时解脱”并非终极涅槃
> “时解脱” (Samayavimutta) 在此通常指通过修习奢摩他（止禅）所获得的世间禅定境界，或是需要外在条件配合才能安住的定力。由于世间定力容易受到外境干扰，如果比丘热衷于杂事、闲聊、嗜睡、群聚，或不如理审察心，定力就会退失。这与佛陀教导的究竟“非时解脱”（无为涅槃、诸漏已尽的阿罗汉果）不退转的特质有着本质的区别。

“诸比丘！有这五种法，会导致适时解脱（时解脱）的比丘退失。是哪五种呢？”[P:149]

1. 喜欢各种杂务事业。
2. 喜欢闲聊。
3. 贪爱睡眠。
4. 喜欢群聚交际。
5. 不如理思惟（审察）自己所体验到的解脱心。

“诸比丘！这五种法，会导致适时解脱的比丘退失。

“诸比丘！有这五种法，能使适时解脱的比丘不退失。是哪五种呢？

1. 不喜欢各种杂务事业。
2. 不喜欢闲聊。
3. 不贪爱睡眠。
4. 不喜欢群聚交际。
5. 如理思惟（审察）自己所体验到的解脱心。

“诸比丘！这五种法，能使适时解脱的比丘不退失。”

## 10 AN 5.150 时解脱者之二经 (Dutiyasamayavimuttasutta)

“诸比丘！有这五种法，会导致适时解脱的比丘退失。是哪五种呢？”[P:150]

1. 喜欢各种杂务事业。
2. 喜欢闲聊。
3. 贪爱睡眠。
4. 不守护感官之门（不防护六根）。
5. 饮食不知节制。

“诸比丘！这五种法，会导致适时解脱的比丘退失。

“诸比丘！有这五种法，能使适时解脱的比丘不退失。是哪五种呢？

1. 不喜欢各种杂务事业。
2. 不喜欢闲聊。
3. 不贪爱睡眠。
4. 守护感官之门（防护六根）。
