> [!abstract] AI 导读
> 本篇包含了《增支部》二集的第二个“五十经篇”，总共分为五个品：人品、乐品、有相品、法品、愚者品。经文通过两两对比的方式，开示了世间不同人物的殊胜与低劣、不同层次快乐的递进关系、恶不善法的生起因缘，以及智者与愚者在对待戒律和佛法上的根本差异。此篇内容简明扼要，直接指向修行者的日常起心动念与实修抉择。

## 1 AN 2.53-64 人品 (Puggalavagga)

### 1.1 AN 2.53 两种出世之人

“比丘们！有两种人出现在世间，是为了大众的利益，为了大众的安乐，为了天人与人类的义利、利益与安乐。是哪两种呢？如来、阿罗汉、正自觉者，以及转轮王。比丘们！这就是为了大众利益、安乐，为了天人与人类的义利、利益与安乐而出现在世间的两种人。”[P:53]

### 1.2 AN 2.54 两种希有示现

“比丘们！这两种人出现在世间，是极为希有的人。是哪两种呢？如来、阿罗汉、正自觉者，以及转轮王。比丘们！这两种人出现在世间，是极为希有的人。”[P:54]

### 1.3 AN 2.55 两种受人追悼之死

“比丘们！有这两种人的离世，会让许多人感到追悼与惋惜。是哪两种人呢？如来、阿罗汉、正自觉者，以及转轮王。比丘们！这两种人的离世，会让许多人感到追悼与惋惜。”[P:55]

### 1.4 AN 2.56 两种配受建塔之人

“比丘们！有这两种人是值得为他们建造佛塔的。是哪两种呢？如来、阿罗汉、正自觉者，以及转轮王。比丘们！这两种人是值得建造佛塔的。”[P:56]

### 1.5 AN 2.57 两种觉者

“比丘们！有这两种觉者。是哪两种呢？如来、阿罗汉、正自觉者，以及独觉佛。比丘们！这就是两种觉者。”[P:57]

### 1.6 AN 2.58 漏尽比丘与良象

“比丘们！有这两者，对轰鸣的雷声不会感到畏惧。是哪两者呢？漏尽比丘，以及良种的大象。比丘们！这两者对轰鸣的雷声不会感到畏惧。”[P:58]

### 1.7 AN 2.59 漏尽比丘与良马

“比丘们！有这两者，对轰鸣的雷声不会感到畏惧。是哪两者呢？漏尽比丘，以及良种的马匹。比丘们！这两者对轰鸣的雷声不会感到畏惧。”[P:59]

### 1.8 AN 2.60 漏尽比丘与狮子

“比丘们！有这两者，对轰鸣的雷声不会感到畏惧。是哪两者呢？漏尽比丘，以及兽王狮子。比丘们！这两者对轰鸣的雷声不会感到畏惧。”[P:60]

### 1.9 AN 2.61 紧那罗不作人语

“比丘们！紧那罗（天人）看到两种道理，因此不说人类的语言。是哪两种呢？‘为了不说虚妄的谎言，又为了不用不实之词去诬陷他人。’比丘们！紧那罗看到了这两种道理，因此不说人类的语言。”[P:61]

### 1.10 AN 2.62 女人的两种不满足

“比丘们！女人直到命终，对两件事依然不会感到厌倦与满足。是哪两件事呢？性交欲乐，以及生育。比丘们！女人直到命终，对这两件事都不会感到厌倦与满足。”[P:62]

> [!note] AI 解析：关于轮回中的贪爱
> 此处的开示直指众生在轮回中最深层的生物性与心理性执取。在佛法中，“性交”与“生育”代表了欲界众生延续生命、繁衍后代的最强烈本能（渴爱）。不仅是女性，众生普遍被这种繁衍的本能所束缚，唯有通过修习四圣谛、断除渴爱，才能从这种深层的执取中解脱。

### 1.11 AN 2.63 与善人及不善人同住

“比丘们！我将为你们宣说与不善人同住，以及与善人同住的情况。你们要仔细听，善加作意，我这就说。”[P:63]

“是的，大德！”比丘们回答世尊。世尊如此说道：

“比丘们！怎样是与不善人同住？不善人又是如何同住的呢？比丘们！在这里，如果有长老比丘这样想：‘长老不应该劝诫我，中腊比丘不应该劝诫我，新学比丘也不应该劝诫我；我也不去劝诫长老，我不去劝诫中腊比丘，我也不去劝诫新学比丘。如果长老怀着不善意而非善意来劝诫我，我就应对他说：不然！并与他对抗，即使我略有所知也不回答他。如果中腊比丘劝诫我……如果新学比丘怀着不善意而非善意来劝诫我，我就应对他说：不然！并与他对抗，即使我略有所知也不回答他。’

“中腊比丘也这样想……新学比丘也这样想：‘长老不应该劝诫我，中腊比丘不应该劝诫我，新学比丘也不应该劝诫我；我也不去劝诫长老，我不去劝诫中腊比丘，我也不去劝诫新学比丘。如果长老怀着不善意而非善意来劝诫我，我就应对他说：不然！并与他对抗，即使我略有所知也不回答他。如果中腊比丘劝诫我……如果新学比丘怀着不善意而非善意来劝诫我，我就应对他说：不然！并与他对抗，即使我略有所知也不回答他。’比丘们！这就是与不善人同住，不善人就是这样同住的。

“比丘们！怎样是与善人同住？善人又是如何同住的呢？比丘们！在这里，如果有长老比丘这样想：‘长老应当劝诫我，中腊比丘应当劝诫我，新学比丘也应当劝诫我；我也应当劝诫长老，我应当劝诫中腊比丘，我也应当劝诫新学比丘。如果长老怀着善意而非不善意来劝诫我，我就应对他说：善哉！绝不与他对抗，我略有所知也应当回答他。如果中腊比丘劝诫我……如果新学比丘怀着善意而非不善意来劝诫我，我就应对他说：善哉！绝不与他对抗，我略有所知也应当回答他。’

“中腊比丘也这样想……新学比丘也这样想：‘长老应当劝诫我，中腊比丘应当劝诫我，新学比丘也应当劝诫我；我也应当劝诫长老，我也应当劝诫中腊比丘，我也应当劝诫新学比丘。如果长老怀着善意而非不善意来劝诫我，我就应对他说：善哉！绝不与他对抗，我略有所知也应当回答他。如果中腊比丘劝诫我……如果新学比丘怀着善意而非不善意来劝诫我，我就应对他说：善哉！绝不与他对抗，我略有所知也应当回答他。’比丘们！这就是与善人同住，善人就是这样同住的。”

> [!tip] AI 小贴士：僧团的“六和敬”精神
> 这段经文展示了佛教僧团互相监督、谦卑受教的核心机制。无论是资深的长老还是刚出家的新学，只要怀着善意，都有责任互相提醒与纠正。拒绝接受他人的忠告，是“不善人”的标志；而欢喜接受批评，则是“善人”的体现。

### 1.12 AN 2.64 止息诤论的功德

“比丘们！当发生诤论时，如果内心充满互相言辞攻击、顽固的邪见、心中的嫌恨、不满与愤怒，且内心未能平息，比丘们！应当预料到这样的结果：‘这将导致长期的、激烈的、凶暴的冲突，比丘们将无法安稳地生活。’[P:64]

“然而，比丘们！当发生诤论时，虽然曾有互相言辞攻击、顽固的邪见、心中的嫌恨、不满与愤怒，但内心能够善加平息，比丘们！应当预料到这样的结果：‘这将不会导致长期的、激烈的、凶暴的冲突，比丘们将能安稳地生活。’”

## 2 AN 2.65-77 乐品 (Sukhavagga)

### 2.1 AN 2.65 在家乐与出家乐

“比丘们！有这两种快乐。是哪两种呢？在家的快乐与出家的快乐。比丘们！这就是两种快乐。在这两种快乐之中，最殊胜的是出家的快乐。”[P:65]

### 2.2 AN 2.66 欲乐与出离乐

“比丘们！有这两种快乐。是哪两种呢？感官欲望的快乐与出离的快乐。比丘们！这就是两种快乐。在这两种快乐之中，最殊胜的是出离的快乐。”[P:66]

### 2.3 AN 2.67 依乐与无依乐

“比丘们！有这两种快乐。是哪两种呢？有执取（依）的快乐与无执取（无依）的快乐。比丘们！这就是两种快乐。在这两种快乐之中，最殊胜的是无执取的快乐。”[P:67]

### 2.4 AN 2.68 有漏乐与无漏乐

“比丘们！有这两种快乐。是哪两种呢？有漏烦恼的快乐与无漏烦恼的快乐。比丘们！这就是两种快乐。在这两种快乐之中，最殊胜的是无漏烦恼的快乐。”[P:68]

### 2.5 AN 2.69 有染乐与无染乐

“比丘们！有这两种快乐。是哪两种呢？世俗物质的快乐（有染乐）与出世间的快乐（无染乐）。比丘们！这就是两种快乐。在这两种快乐之中，最殊胜的是无染的快乐。”[P:69]

### 2.6 AN 2.70 圣乐与非圣乐

“比丘们！有这两种快乐。是哪两种呢？圣者的快乐与非圣者的快乐。比丘们！这就是两种快乐。在这两种快乐之中，最殊胜的是圣者的快乐。”[P:70]

### 2.7 AN 2.71 身乐与心乐

“比丘们！有这两种快乐。是哪两种呢？身体的快乐与心理的快乐。比丘们！这就是两种快乐。在这两种快乐之中，最殊胜的是心理的快乐。”[P:71]

### 2.8 AN 2.72 有喜乐与无喜乐

“比丘们！有这两种快乐。是哪两种呢？伴随喜悦的快乐与不伴随喜悦的快乐。比丘们！这就是两种快乐。在这两种快乐之中，最殊胜的是不伴随喜悦的快乐。”[P:72]

### 2.9 AN 2.73 悦乐与舍乐

“比丘们！有这两种快乐。是哪两种呢？愉悦的快乐与平等舍的快乐。比丘们！这就是两种快乐。在这两种快乐之中，最殊胜的是平等舍的快乐。”[P:73]

### 2.10 AN 2.74 定乐与非定乐

“比丘们！有这两种快乐。是哪两种呢？有三摩地（定）的快乐与无三摩地的快乐。比丘们！这就是两种快乐。在这两种快乐之中，最殊胜的是有三摩地的快乐。”[P:74]

### 2.11 AN 2.75 有喜缘乐与无喜缘乐

“比丘们！有这两种快乐。是哪两种呢？以喜悦为所缘的快乐与不以喜悦为所缘的快乐。比丘们！这就是两种快乐。在这两种快乐之中，最殊胜是不以喜悦为所缘的快乐。”[P:75]

### 2.12 AN 2.76 悦缘乐与舍缘乐

“比丘们！有这两种快乐。是哪两种呢？以愉悦为所缘的快乐与以平等舍为所缘的快乐。比丘们！这就是两种快乐。在这两种快乐之中，最殊胜的是以平等舍为所缘的快乐。”[P:76]

### 2.13 AN 2.77 色缘乐与无色缘乐

“比丘们！有这两种快乐。是哪两种呢？以色法为所缘的快乐与以无色法为所缘的快乐。比丘们！这就是两种快乐。在这两种快乐之中，最殊胜的是以无色法为所缘的快乐。”[P:77]

> [!success] AI 阶段性回顾：快乐的升华次第
> 这一品的十三部短经，清晰地描绘了佛教中“快乐”（Sukha）的升华路线：从世俗感官的追求，转向出世间的禅定与解脱。在禅修体验中，越是宁静、微细、不依赖外物（无依、无漏、无喜、舍、无色）的快乐，其境界就越殊胜、越持久。

## 3 AN 2.78-87 有相品 (Sanimittavagga)

### 3.1 AN 2.78 有相的恶不善法

“比丘们！恶不善法是因为有相（特征/根源）而生起的，并非无相而生。只要断除了那个相，那些恶不善法就不会再存在。”[P:78]

### 3.2 AN 2.79 有因缘的恶不善法

“比丘们！恶不善法是因为有因缘（绪）而生起的，并非无因缘而生。只要断除了那个因缘，那些恶不善法就不会再存在。”[P:79]

### 3.3 AN 2.80 有因的恶不善法

“比丘们！恶不善法是因为有因而生起的，并非无因而生。只要断除了那个因，那些恶不善法就不会再存在。”[P:80]

### 3.4 AN 2.81 有行的恶不善法

“比丘们！恶不善法是因为有行（造作）而生起的，并非无行而生。只要断除了那些行，那些恶不善法就不会再存在。”[P:81]

### 3.5 AN 2.82 有缘的恶不善法

“比丘们！恶不善法是因为有缘而生起的，并非无缘而生。只要断除了那个缘，那些恶不善法就不会再存在。”[P:82]

### 3.6 AN 2.83 有色的恶不善法

“比丘们！恶不善法是因为有色法而生起的，并非无色法而生。只要断除了那个色法，那些恶不善法就不会再存在。”[P:83]

### 3.7 AN 2.84 有受的恶不善法

“比丘们！恶不善法是因为有感受（受）而生起的，并非无受而生。只要断除了那个受，那些恶不善法就不会再存在。”[P:84]

### 3.8 AN 2.85 有想的恶不善法

“比丘们！恶不善法是因为有想而生起的，并非无想而生。只要断除了那个想，那些恶不善法就不会再存在。”[P:85]

### 3.9 AN 2.86 有识的恶不善法

“比丘们！恶不善法是因为有识而生起的，并非无识而生。只要断除了那个识，那些恶不善法就不会再存在。”[P:86]

### 3.10 AN 2.87 有为缘的恶不善法

“比丘们！恶不善法是因为有为法（条件造作）为所缘而生起的，并非无有为法为所缘而生。只要断除了那些有为法，那些恶不善法就不会再存在。”[P:87]

> [!important] AI 法义精髓：因缘生灭的法则
> 此品反复强调了佛法的核心原则：**此有故彼有，此灭故彼灭**。一切烦恼与不善（如贪嗔痴）都不是凭空产生的，而是依赖于特定的相、因、缘、五蕴（色受想行识）而存在。只要在禅修中看透并拔除这些根源，烦恼自然瓦解。

## 4 AN 2.88-98 法品 (Dhammavagga)

### 4.1 AN 2.88 心解脱与慧解脱

“比丘们！有这两种法。是哪两种呢？心解脱与慧解脱。比丘们！这就是两种法。”[P:88]

> [!note] AI 解析：心解脱与慧解脱
> **心解脱 (Cetovimutti)**：主要指通过修习奢摩他（止/定），使心远离贪欲的烦恼束缚。
> **慧解脱 (Paññāvimutti)**：主要指通过修习毗婆舍那（观/智慧），如实照见无常、苦、无我，从而断除无明的烦恼束缚。两者的结合是证得阿罗汉果的必要条件。

### 4.2 AN 2.89 策励与不散乱

“比丘们！有这两种法。是哪两种呢？精勤策励与心不散乱。比丘们！这就是两种法。”[P:89]

### 4.3 AN 2.90 名与色

“比丘们！有这两种法。是哪两种呢？名法（精神）与色法（物质）。比丘们！这就是两种法。”[P:90]

### 4.4 AN 2.91 明与解脱

“比丘们！有这两种法。是哪两种呢？明（智慧）与解脱。比丘们！这就是两种法。”[P:91]

### 4.5 AN 2.92 有见与无有见

“比丘们！有这两种法。是哪两种呢？有见（常见）与无有见（断见）。比丘们！这就是两种法。”[P:92]

### 4.6 AN 2.93 无惭与无愧

“比丘们！有这两种法。是哪两种呢？无惭与无愧。比丘们！这就是两种法。”[P:93]

### 4.7 AN 2.94 惭与愧

“比丘们！有这两种法。是哪两种呢？惭（对造恶感到自我羞耻）与愧（对造恶感到畏惧他人谴责与果报）。比丘们！这就是两种法。”[P:94]

### 4.8 AN 2.95 恶言与恶友

“比丘们！有这两种法。是哪两种呢？恶言（难以受教）与结交恶友。比丘们！这就是两种法。”[P:95]

### 4.9 AN 2.96 善言与善友

“比丘们！有这两种法。是哪两种呢？善言（容易受教）与结交善友。比丘们！这就是两种法。”[P:96]

### 4.10 AN 2.97 界善巧与作意善巧

“比丘们！有这两种法。是哪两种呢？界善巧（了知界法）与作意善巧（如理作意）。比丘们！这就是两种法。”[P:97]

### 4.11 AN 2.98 犯罪善巧与出罪善巧

“比丘们！有这两种法。是哪两种呢？对犯戒的善巧（了知犯戒）与对出罪的善巧（了知如何忏悔恢复清净）。比丘们！这就是两种法。”[P:98]

## 5 AN 2.99-118 愚者品 (Bālavagga)

### 5.1 AN 2.99 愚者对待重担

“比丘们！有这两种人是愚者。是哪两种呢？第一种是，时机未到却去挑起重担的人；第二种是，时机已到却不去承担重担的人。比丘们！这两种人是愚者。”[P:99]

### 5.2 AN 2.100 智者对待重担

“比丘们！有这两种人是智者。是哪两种呢？第一种是，时机未到就不去挑起重担的人；第二种是，时机已到就勇于承担重担的人。比丘们！这两种人是智者。”[P:100]

### 5.3 AN 2.101 愚者颠倒许可与不许可
